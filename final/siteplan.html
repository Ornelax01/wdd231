Board Game Vault Site Plan

Site Name

Board Game Vault

I selected this name because the website will serve as a collection of board game information, recommendations, ratings, and game details. The word "Vault" represents a place where visitors can discover and explore a large variety of board games.

Optional Domain: boardgamevault.com

Site Purpose

Board Game Vault will provide information about board games, including game descriptions, player counts, play times, difficulty levels, and recommendations. Visitors will be able to browse a collection of games and submit a form requesting personalized board game recommendations.

Scenarios

  • What board game would be best for a family game night with children and adults?
  • Which strategy games are recommended for beginner players?
  • What games support large groups of six or more players?

Color Scheme

Primary Color - #1d3557

This color will be used for headings, the navigation bar, header, footer, and major page sections.

Secondary Color - #f4a261

This color will be used for buttons, accents, highlights, and section borders.

Background Color - #f8f7f2

This color will be used as the page background to provide a clean and readable appearance.

Typography

Roboto will be used for body text because it is clean, modern, and easy to read on mobile and desktop devices.

Georgia will be used for headings because it creates visual contrast and helps section titles stand out.
